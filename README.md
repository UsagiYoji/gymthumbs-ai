This scaffold contains a minimal Next.js 14 frontend and a worker scaffold to run local demos.

Instructions:
1. Copy .env.example -> .env and fill keys
2. cd apps/web && npm install
3. cd ../../apps/worker && npm install
4. Run dev:
   - npm run dev (in apps/web)
   - node worker.js (in apps/worker)

Deploy frontend to Vercel and worker to Render/Cloud Run if you want production.
